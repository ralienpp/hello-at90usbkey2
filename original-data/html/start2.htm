You can now use your key as a small development platform for AVR firmware and quickly download your own programs directly from the USB interface. Please refer to the manual for hardware details and possibly connecting a JTAG emulator. For developping mass storage applications needing local access and safe access to the file structure you may preffer not using Windows® File Allocation Table (FAT) systems. Hcc-embedded provides alternate file systems.

If you want to add USB connection to your AVR application, please register (code AVRKEY) to accept our royalty free license terms and customize source code to your needs. We currently provide files for IAR and WinAVR compilers. You can start with the mouse project and get familiar with AVRUSB. Then you can reprogram the original binary code, the DataFlash should have kept its content.

New features and demonstration software will be added with updates from our web site.
You may also share experience on the AVR forum and try software from 3rd parties.

We hope you will have a great time with your AVR USB key.
