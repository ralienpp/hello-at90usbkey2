You may now wish to try other features before going into more detail about the mass storage function.

You can use the joystick as a mouse, which simply moves the cursor on your computer.

The third function provided by your AVR USB Key is a generic HID IO interface. To access it you must install and run this software "UsbHidGenericDemo.exe" on a PC*. Click on "Connect" in the generic HID window and it provides temperature report, LED control and Joystick input. The potentiometer is only supported by the STK525 AVR USB Starter Kit.

To avoid any warning and possible disk corruption if you have modified its content, you must perform a "safe removal" before disconnecting, resetting or starting Device Firmware Upgrade (DFU). Your key is a composite device and a hotfix is required on Windows 2000 platforms to force synchronization with it, no fix is required on later OS versions.

Without changing firmware, you can disconnect your key from your computer, power it with a 9V battery with the battery clip and connect a plain USB mouse or another AVR USB key with the host adapter. It is now an HID embedded host and moving the mouse has its LEDs blinking.

All this is outlined in the quickstart guide.

note: Win2000 or later
